Automated surface-defect detection for aircraft inspection.

Aircraft Inspector identifies missing fastener heads, paint loss, corrosion, and scratches on aircraft skin panels from a single photograph. Detection is performed by an on-device model — no image is transmitted over a network.

Section 01

Defect classification

ClassDescription
Missing head Fastener head sheared off or absent from a rivet line.
Paint-off Coating loss exposing the underlying substrate.
Rust Corrosion present on an exposed metal surface.
Scratch Linear abrasion across a panel surface.

Section 02

Operating procedure

  1. Install

    Run the NSIS installer. Administrator privileges are not required — the application installs to the current user profile.

  2. Load a photograph

    Provide an image of the panel or fuselage section to be inspected, from a file or a connected camera.

  3. Review the output

    Detected defects are outlined on the image and labeled with class and confidence score.
